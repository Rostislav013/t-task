How to create a branch
1. go to master git checkout master
2. Check if u r there: git branch  (*master)
3. Create branch: git branch urBranchName
4. Go to ur branch: git checkout urBranchName
5. check if u r there now: git branch  (*urBranchName)

Delete a Local GIT branch
1. git checkout master (need to move out of ur branch)
2. git branch -D branch_name
3. git branch (branch will not be shown ib branch lists)
